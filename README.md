# SiFM - Simple File Manager

SiFM is an open-source, web-based PHP file manager that allows you to efficiently manage your files and folders through a simple interface. It is designed to be lightweight, user-friendly, and versatile, meeting the needs of developers and users alike.

## Features

### File Management Actions

- **View Files and Folders**: Navigate through directories to view their contents, including details like size, permissions, and last modified date.
  
- **Create Files and Folders**: Easily create new files or folders within your directories to keep your structure organized.
  
- **Edit Files**: Modify the content of text files directly in the browser using a built-in text editor. Supports syntax highlighting for various file types.

- **Upload Files**: Select and upload files from your local machine to the server, with support for multiple files at once.

- **Download Files**: Download individual files or multiple files as a ZIP archive for convenience.

- **Copy, Move, and Rename**: Duplicate, relocate, or rename files and folders as needed, ensuring flexibility in organizing your data.

- **Delete Files and Folders**: Permanently remove files or folders with confirmation to prevent accidental loss of important data. Deleted items cannot be recovered.

### Advanced Features

- **Permissions Management**: Change file and folder permissions (chmod) to control access, essential for multi-user environments.

- **Backup and Restore**: Create ZIP backups of files and folders, offering peace of mind and protection against accidental deletion or corruption.

- **Extract Files**: Unzip files and folders to access their contents easily, making it convenient to manage compressed files.

- **Search Functionality**: Quickly locate files and folders using keyword searches and wildcards, saving time in large collections.

- **PHP Syntax Checking**: Verify the syntax of PHP files to identify and fix errors directly within the SiFM interface.

- **Comprehensive File Information**: Access detailed information about files, including size, MIME type, modification date, and ownership.

## Installation

1. **Download SiFM**: Download the `sifm.php` file from the official source.
2. **Upload to Server**: Place `sifm.php` in the document root of your web server or in any directory accessible via a browser.
3. **Configure Settings**: Open the `sifm.php` file in a text editor and adjust the configuration options as required.
4. **Access the Interface**: Open your browser and navigate to `http://yourdomain.com/sifm.php` to access SiFM.

## Configuration

You can customize various settings in the `sifm.php` configuration section:

- **SiFM Name**: Change the application name.
- **Version**: Set the SiFM version.
- **Authentication**: Enable or disable user authentication.
- **Admin Credentials**: Set the admin username and password (hashed for security).
- **Home Folder**: Specify the base directory for file operations.
- **Timezone**: Set the timezone for date and time display.

## Usage

After installation, you can start managing your files through the SiFM interface. Features such as uploading files, creating folders, and performing file operations are straightforward with easily accessible buttons.
